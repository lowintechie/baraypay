# baraypay

All-in-one BarayPay package.

Install once:

```bash
npm install baraypay
```

Use unified API + providers from one import:

```ts
import { createBarayPay, abaPayway, bakongKHQR } from "baraypay";
```

Or optional subpath imports:

```ts
import { abaPayway } from "baraypay/abapayway";
import { bakongKHQR } from "baraypay/bakong";
```
